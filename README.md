# Process App

A Vite + React + TypeScript dashboard for visualizing processes as countdowns, recurring cycles, and long-range milestones.

## What it shows

- Short cycles such as minute, hour, day, month, and year countdowns
- Calendar milestones like holidays and moon phases
- Deep-time events such as decades, centuries, and cosmic horizons

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
